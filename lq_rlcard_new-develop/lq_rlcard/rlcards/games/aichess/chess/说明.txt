todo: 算法流程
    1.读取现有棋盘
    2.遍历棋盘
    3.遍历棋子
    4.可行的走法与能救助己方棋子的走法保存至一个list中
    5.返回该list
    ```
    def generate_move(self, who):
        """
        返回所有可行的走法，一个step类型的list
        """
        res_list = []
        for x in range(9):
            for y in range(10):
                if self.board[x][y].chess_type != 0:
                    if self.board[x]][y].belong != who:
                        continue
                    # 返回每个棋子的走法
                    list2 = self.get_chess_move(x, y, who)
                    res_list = res_list + list2
        return res_list
    ```
