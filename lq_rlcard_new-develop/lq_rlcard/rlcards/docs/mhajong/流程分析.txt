TODO: 麻将血流红中捡漏玩法(开局到结束)
1.麻将开局
    1.1 四位玩家，每人各执骰子一次，点数最大为庄家(相同点数，通过排序取首位即可)
    1.2 每位玩家发12张牌+1张红中，庄家开局作为首位出牌者，进行摸牌(开局庄家持牌: 14)
    1.3 庄家摸牌后，先判断庄家是否能够天胡(庄家只判断天胡，不判断天听),
    1.4 庄家不能天胡，则计算庄家手中的顺子、将牌、刻子(庄家之后可以进行碰操作和杠操作)

2.庄家出牌
    2.1 庄家出牌之后，判断庄家是否符合天听状态
    2.2 庄家操作完之后，判断其他玩家是否要得起这张牌
    2.3 判断其他玩家是否能够(碰、杠、胡)
        2.3.1 当有玩家碰、杠时，下一位玩家则为进行(碰/杠玩家ID + 1)，进行下一次操作
        2.3.2 下一位玩家进行摸牌，判断是否能够胡，如果玩家是第一次出牌，则判断玩家是否能够天听
        2.3.3 当玩家不能胡，也不是天听状态时，判断玩家手中的顺子、将牌、刻子
        2.3.4 完成操作之后，出牌
        2.3.5 判断其他玩家是否能够(碰、杠、胡)

3.下家操作
    3.1 庄家的下一位出牌玩家 = (庄家ID + 1) % 当局游戏人数
    3.2 下家摸牌，当下架为首次摸牌，且没有进行任何操作时，判断玩家是否能胡，如果不能胡，则判断玩家是否符合天听状态
    3.3 当玩家不能胡，也不是天听状态时，判断玩家手中的顺子、将牌、刻子
    3.4 完成操作之后，出牌
    3.5

4.下下家
    4.1 庄家的下下一位出牌玩家 = (庄家ID + 2) % 当局游戏人数
    4.2 下家摸牌，当下架为首次摸牌，且没有进行任何操作时，判断玩家是否能胡，如果不能胡，则判断玩家是否符合天听状态
    4.3 当玩家不能胡，也不是天听状态时，判断玩家手中的顺子、将牌、刻子
    4.4 完成操作之后，出牌
    4.5 判断其他玩家是否能够(碰、杠、胡)

5.最后玩家
    5.1 庄家的下下一位出牌玩家 = (庄家ID + 3) % 当局游戏人数
    5.2 下家摸牌，当下架为首次摸牌，且没有进行任何操作时，判断玩家是否能胡，如果不能胡，则判断玩家是否符合天听状态
    5.3 当玩家不能胡，也不是天听状态时，判断玩家手中的顺子、将牌、刻子
    5.4 完成操作之后，出牌
    5.5 判断其他玩家是否能够(碰、杠、胡)

6.(一圈结束)所有人出完一次牌之后，将不再判断玩家天听情况

7.庄家摸牌，继续执行(2-5步骤)，直到玩家胡开则游戏结束

8.游戏结束，进行算法，赢家为下一局庄家

9.根据对局玩家操作设置对应奖励

10.将玩家状态信息和奖励数据保存，传入网络中

明杠: 其他人打出一张牌，自己手中有三张相同的牌，即可杠牌，称为明杠(补一张牌)
转弯杠: 进行过碰牌，形成三张明面的刻子，自己再摸到此牌，则为转弯杠/索杠(补一张牌)
暗杠: 玩家手中存在三张相同的，未明面，再摸到此牌时，则为暗杠，暗杠后要将四张牌都扣倒，然后补一张牌继续打
憨包杠: 起手牌就存有四张相同的，憨包杠在可以杠时没有立即杠，之后再杠，只算憨包杠
